# 클라이언트
중요한 파일은
canvas.js
canvasContext.js
app.js
clearCanvasButton.js
입니다.

npm start하시고, 추가로 localhost 3000 페이지를 하나 더 띄워서
왼쪽 오른쪽으로 나눠 띄우시고, 왼쪽에 그림을 그리면 동기화되어 오른쪽에도 나타납니다.
clear버튼도 한쪽에서 누르면 다른 캔버스도 지워집니다.
